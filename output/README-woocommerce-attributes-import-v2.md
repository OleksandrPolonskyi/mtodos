# Імпорт атрибутів у WooCommerce (v2)

Файл згенеровано через скіл `woocommerce-import-csv` (скрипт `build_woocommerce_csv.py`) і адаптовано під ваш кейс атрибутів.

## Основний файл
- `woocommerce-attributes-seed-v2.csv`

## Fallback, якщо імпорт "висить"
- `woocommerce-attributes-seed-v2-split.csv`

## Що імпортується
- Атрибут 1: `Колір` (45 значень)
- Атрибут 2: `Розмір` (`ONE SIZE`, технічний)
- Атрибут 3: `Матеріал` (32 значення)
- `Денім` у матеріалах злитий у `Джинс`

## Порядок
1. Відкрий `Товари -> Атрибути` і перевір, що атрибути `Колір` та `Матеріал` існують.
2. Відкрий `Товари -> Імпорт`.
3. Спочатку завантаж `woocommerce-attributes-seed-v2.csv`.
4. Якщо імпорт довго не завершується, скасуй і завантаж `woocommerce-attributes-seed-v2-split.csv`.
5. Після імпорту перевір терміни у `Товари -> Атрибути -> Налаштувати терміни`.
6. Видали технічні товари з SKU, що починаються на `MD-ATTR-SEED-20260206-`.
