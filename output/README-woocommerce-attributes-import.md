# Імпорт кольорів і матеріалів у WooCommerce

## Файли
- `woocommerce-seed-attributes-colors-materials.csv` — головний файл для імпорту (UA заголовки).
- `woocommerce-seed-attributes-colors-materials-en.csv` — fallback-файл (EN заголовки).
- `woocommerce-attribute-terms-reference.csv` — довідник усіх термінів, які будуть додані.
- `woocommerce-material-alias-map.csv` — нормалізація матеріалів (`Денім` -> `Джинс`).

## Як імпортувати
1. У WooCommerce відкрий `Товари -> Атрибути` і створи (або перевір наявність) глобальних атрибутів:
   - `Колір`
   - `Матеріал`
2. Відкрий `Товари -> Імпорт` і завантаж файл `woocommerce-seed-attributes-colors-materials.csv` (або `woocommerce-seed-attributes-colors-materials-en.csv`, якщо мапінг колонок зручніший).
3. На етапі мапінгу колонок перевір, що атрибути зіставились коректно (`Назва 1 атрибуту`, `1 значення атрибуту`, тощо).
4. Запусти імпорт. Після цього терміни з'являться в `Налаштувати терміни` для обох атрибутів.
5. Технічний товар `MD-ATTR-SEED-20260206` можна видалити після перевірки.

## Примітка
У матеріалах виконана нормалізація: `Денім` об'єднано в `Джинс`.
